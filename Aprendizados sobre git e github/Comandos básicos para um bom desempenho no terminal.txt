Comandos básicos

Para mostrar o diretório que está:
dir



Para navegar entre as pastas:

cd / (eu entro na pasta primária do sistema

cd Windows (eu entro na pasta do windows)

cd .. (volto uma pasta)

Para limpar o terminal:
cls ou clear

Para criar uma pasta

mkdir workspace
